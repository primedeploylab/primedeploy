## Troubleshooting

### Logo not showing?
- Check URL is correct and accessible
- Make sure image format is supported (PNG, JPG, SVG)
- Try uploading to different host
- Clear browser cache

### Social icons not appearing?
- Make sure URLs are complete (include https://)
- Check spelling of platform names
- Save settings after adding
- Refresh page

### Changes not visible?
- Click "Save Settings" button
- Wait for success message
- Refresh your website
- Clear browser cache (Ctrl+F5)

### Logo too big/small?
- Logo height is auto-limited to 50px in navbar
- Use image editing tool to resize
- Recommended: 200x60px
